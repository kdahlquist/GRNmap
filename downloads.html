Download GRNmap Executable

We recommend using the latest version of the GRNmap executable. Links to download prior versions are provided for archival purposes.

System Requirements and Compatability

  • You must have administrator rights or have the MATLAB Runtime Compiler already installed in your machine to use the stand-alone executable.
  • GRNmap is only compatible with the Windows operating system because of the function it uses to read and write Microsoft Excel spreadsheets. It has only been tested on Windows 7 and 10. The executable does not function on Windows 11.
  • We recommend running GRNmap with a minimum of 8.00 GB of RAM and a 2.40 GHz processor. GRNmap may be compatible with slower systems; the amount of RAM and processor speed will affect the speed with which GRNmap completes the estimation.
